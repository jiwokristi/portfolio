# jiwokristi.dev

Personal portfolio site built with Next.js, Tailwind CSS, and GSAP.

## Tech Stack

- **Framework:** Next.js 16 (App Router, Turbopack)
- **Styling:** Tailwind CSS 4 with custom design tokens
- **Animation:** GSAP + ScrollTrigger
- **Theming:** next-themes (dark/light/system)
- **Components:** CVA (class-variance-authority)
- **Fonts:** Inter, JetBrains Mono

## Pages

- `/` — Home with hero and project grid
- `/about` — Philosophy, evidence, stack, values
- `/resume` — Interactive resume with PDF download
- `/projects/[slug]` — Case study pages

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
